# График урона Танки Онлайн

[English](./README.md) │ [简体中文](./README_zh.md) │ [Русский](./README_ru.md)

Небольшой инструмент на React для сравнения урона башен Танки Онлайн в фиксированном 10-секундном интервале.

График использует реальные моменты атак башен из локальной модели данных, а не вычисляет урон через фиксированные интервалы времени. Можно выбрать несколько башен одновременно, при этом каждая выбранная башня отображается в виде ступенчатой линии накопленного урона.

## Возможности

- Выбор нескольких башен и сравнение их накопленного урона.
- Сохранение выбранных башен в URL с помощью параметра запроса `turrets`.
- Отображение реальных моментов атак, включая значения вроде `1.1s`, `2.2s` и `3.3s`.
- Поддержка повторяющихся циклов перезарядки с маркерами `-1` и `-2`.
- Поддержка независимых событий дополнительного урона через `add`.
- Включение и отключение всплывающих подсказок графика.
- Применение множителя урона с задержкой debounce.
- Отображение опорных линий для значений защиты лёгких, средних и тяжёлых корпусов.

## Модель урона

Данные башен находятся в файле `src/turrets.ts`.

Каждая башня содержит массив `reloadTime`. Его значения интерпретируются как относительные интервалы перед каждой атакой, а не как абсолютные отметки времени.

Примеры:

```ts
;[0, 1.1, -1]
```

Первая атака происходит в `0s`, после чего последний обычный интервал повторяется бесконечно: `1.1s`, `2.2s`, `3.3s` и так далее.

```ts
;[1.2, 3.7, -1]
```

Первая атака происходит через `1.2s`, после чего интервал `3.7s` повторяется бесконечно.

```ts
;[0, 1, 2, -2]
```

После начальной последовательности повторяется весь цикл интервалов. Если последовательность начинается с `0`, первый `0` означает, что башня изначально заряжена, и в последующих циклах он не повторяется.

`add` определяет независимые события дополнительного урона:

```ts
add: [intervalPattern, additionalDamage]
```

`intervalPattern` использует те же правила `-1` и `-2`, что и `reloadTime`. Каждое дополнительное событие увеличивает накопленный урон на `additionalDamage`.

## Структура проекта

```txt
src/
  features/
    chart.tsx       Отрисовка графика и генерация данных урона
    control.tsx     Управление подсказками и множителем урона
    grid.tsx        Сетка выбора башен
  layout/
    main.tsx        Основное состояние приложения и синхронизация с URL
  turrets.ts        Данные урона, таймингов и отображения башен
```
